## 🔧 如何繼續開發

### 啟動專案
```bash
cd autopass-demo
npm run dev
```
然後打開瀏覽器訪問顯示的網址（通常是 http://localhost:5173）

### 添加新頁面的步驟
1. **準備Figma設計稿**
2. **使用Figma get code工具**提取代碼
3. **在src/components/建立新組件**
4. **修改Navigation.tsx**更新路由邏輯
5. **在App.tsx**中添加頁面路由

### 常見問題解決
- **RWD問題**：檢查CSS的width設定和AppShell配置
- **樣式問題**：記得在index.css添加全域樣式覆蓋
- **Logo問題**：Logo檔案放在public/資料夾中
